# SysInfo v1.5.1

## Summary

This release finalizes the quality hardening work with tracked release checksums and CI validation for those checksums.

## Verification

Use the checksum file at `checksums/v1.5.1.sha256` to verify release files:

```sh
sha256sum -c checksums/v1.5.1.sha256
```

On systems without `sha256sum`, use:

```sh
shasum -a 256 system-info.sh install-system-info.sh README.md
```
